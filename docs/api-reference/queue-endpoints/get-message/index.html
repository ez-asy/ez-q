Get Message

Pull a message from a queue.

Request​

GET /queues/{queue_name}/message/

Url Path Parameters

queue_name
string
The name of the queue you wish to pull a message from.

Response​

  • 200
  • 500
A JSON containing the id of the message popped and it's content.

Response Body

id
string
The ID of the message that has been popped
content
any json
JSON content of the message pulled - the same content defined when the message was published

Example​

{
  id: "35548190-bad1-4626-95cf-767e0fdf0e66",
  content: {
    "name": "Ada Lovelace",
    "email": "ada@geemail.com",
    "bio": "First programmer. No big deal.",
    "age": 198
  }
}
An unexpected error has happened on the server.
